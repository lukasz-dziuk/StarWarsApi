## Description

This is Star Wars API. You can do CRUD operations on characters from Star Wars.

API documentation address: http://localhost:3000/api/
## Installation

At the beginnings make sure your mongoDB is up and running. Please read README in mongoDB folder.

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

```
